# Агент для работы с плагином

**Плагин:** https://github.com/modelcontextprotocol/servers/tree/main/src/sequentialthinking

Проверить, что MCP сервер зарегистрирован
`opencode mcp list`

Должен быть sequential-thinking в списке

Запустить агента
`opencode --agent thinker "Реши: 25 * 4 + (18 - 6) / 2"`
